            end

            %% Connections
            User -->|Uploads PDF| Router
            Router --> Parser
            Parser -->|Raw Text| Regex
            Regex -->|Skill Vector| Mapper
            Mapper -->|Matched Roles| GapAlgo
            
            GapAlgo -->|Missing Skills| Threads
            GapAlgo -->|Context| Gemini
            
            Threads -->|Parallel Requests| SerpApi
            SerpApi -->|Job Market Data| Threads
            SerpApi -->|Tutorial Links| Gemini
            
            Gemini -->|Cover Letter / Learning Plan| Router
            Router -->|Final Dashboard| User
