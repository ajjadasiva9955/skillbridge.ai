SkillBridge.AI : 

🚀 AI-Powered Resume Job Matcher & Career Agent (RJM-Agent)

RJM-Agent is an intelligent career assistant that analyzes resumes, matches them with relevant job roles, identifies skill gaps, and generates personalized learning plans and AI-powered cover letters.

The system combines rule-based skill extraction with agentic AI to provide real-time job recommendations, market insights, and guided upskilling pathways.

🔑 Features
📄 Resume parsing (PDF)
🎯 Skill extraction & job role matching
📊 Skill gap analysis
🤖 AI-generated cover letters
📚 Personalized learning plans
📈 Real-time job listings & market demand
⚙️ Tech Stack
Python (Flask)
Gemini API (LLM)
SerpAPI (Job data)
HTML, CSS, JavaScript
